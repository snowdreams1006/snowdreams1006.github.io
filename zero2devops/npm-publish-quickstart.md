# npm更新发布插件体验

```
npm config set registry https://registry.npm.taobao.org

npm config set registry https://registry.npmjs.org
```

```bash
npm login
```

> `npm notice Log in on http://registry.npm.taobao.org/`

```bash
npm whoami
```

```bash
npm publish 
```
