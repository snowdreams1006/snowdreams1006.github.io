## 特点

- 面向初学者,适合零基础入门;
- 面向常用操作,技能实用性强;
- 情景教学,理清事情来龙去脉;
- 章节重点知识小结,精华集锦;

## 要求

- 保持哲学三问,是什么,为什么,怎么样?希望最终能有你自己的答案;
- 好记性不如烂笔头,亲自动手操作一遍,你会发现你的理解更上一层楼;
- 授人以鱼不如授人以渔,希望带给你不仅仅是知识更多的是学习的方法;
- 知识重在分享才有价值,鼓励知识传播与分享,创造收益更有价值;
- 因本人能力有限,如有出入,敬请指正,请联系我 [snowdreams1006](https://github.com/snowdreams1006/)
