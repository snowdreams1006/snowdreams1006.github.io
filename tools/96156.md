## 脚本

![](/assets/picgo/d07e2269b8137e27017017b28cffc7f3.png)

第一步: 登录小程序抓取 `x-token`

> 抓取`ylapi.luckystarpay.com`域名下的http请求头`x-token`,类似于下面格式:
>
> #小程序://北京96156/Vr5W4S8igN7IJif

```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.xxx.xxx
```

![](/assets/picgo/dbabde0595a06e6cab61a453524bee55.png)

第二步: 登录网站抓取`Authorization`

作者本人为了实验需要搭建了一款ai问答机器人,可以根据问题和选项,自动给出正确答案,有开发能力的人也可以自行替换成类似项目.

![](/assets/picgo/694a4aa17aab43063faceac16c18b8a4.png)

> 抓取`eladmin.snowdreams1006.cn`域名下http请求头`Authorization`,类似于下面格式:
>
> 注册链接: [https://eladmin.snowdreams1006.cn/register](https://eladmin.snowdreams1006.cn/register)

```
Bearer eyJhbGciOiJIUzUxMiJ9.xxx.xxx
```

![](/assets/picgo/c7b49f5dac4b6b1384e51f2812c19614.png)

第三步: 设置微信推送提醒`wxpusher`

自己给自己发消息,直接扫描下面二维码即可获取你的推送SPT:

![](/assets/picgo/eb9c95ee839e298224da2c4df37b8564.png)

```
SPT_xxx
```

> WxPusher: [https://wxpusher.zjiecode.com/docs/#/](https://wxpusher.zjiecode.com/docs/#/)
