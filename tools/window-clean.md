## 前言

最近电脑部分软件总是发生闪退现象,一看磁盘空间吓了一跳,飘红的 C 盘提示该清理电脑垃圾了.

如果磁盘分区不合理,安装软件时系统默认路径,这些操作都有可能导致 C 盘爆满,下面就清理一下吧!

![](/assets/picgo/440d77017f64de26195b9c529a600e6d.png)

## 1. 清理临时文件

> 临时文件可放心大胆手动删除,默认位于`C:\Users\Administrator\AppData\Local\Temp`目录

- 1. **打开临时文件夹**: 按 【`Win + R`】组合键打开运行对话,输入`%temp%`并按回车【`Enter`】.

![](/assets/picgo/a2b0de76f28b8fc7171b3890cad33db7.png)

- 2. **删除临时文件**: 在打开的文件夹中，按【`Ctrl + A`】选择所有文件和文件夹,先按下【`Shift`】键并保持按住,然后再按下【`Delete`】键进行永久删除临时文件.

![](/assets/picgo/29e89675b3c453b557d69f12b893bca0.png)

删除临时文件中出现**文件正在使用**,先勾选"为所有当前项目执行此操作(A)",再点击跳过(S)即可继续删除.

![](/assets/picgo/5ffa3b310087029d532d812ec9a1a010.png)

- 3. **立即删除不进入回收站**: 清空回收站并设置立即删除.

![](/assets/picgo/bb2661f7425a16937d48eb48946bdfd8.png)

勾选`不将文件移到回收站中,移出文件后立即将其删除`,然后点击**应用**并**确定**,完成设置.

![](/assets/picgo/c3aa0339b690c8f4ab1fc7dce698585a.png)

## 2. 使用内置磁盘清理工具

> Windows 内置的磁盘清理工具可以帮助你清理不必要的系统文件和垃圾文件.

- 1. **打开磁盘清理工具**: 按【`Win + R`】组合键打开运行对话框,输入 `cleanmgr` 并按回车

![](/assets/picgo/f376b24edee587eb230364ea7b2b5c51.png)

- 2. **选择驱动器**: 默认选择 `C` 盘,然后点击【确定】

![](/assets/picgo/4200c5423ab1f894831b882f7876d320.png)

- 3. **选择要删除的文件**：在磁盘清理窗口中,勾选全部项目后点击【确定】

![](/assets/picgo/f57afd57a6cc6a3ffcc216a12ecb3c55.png)

稍后弹出的永久删除确认框中点击【确定】,等待自动清理磁盘.

![](/assets/picgo/9e2feece70da86a3aacbda233df8a0af.png)

删除过程中也可以中途取消,建议什么都不做,让它慢慢删除,整个过程大概也就一两分钟.

![](/assets/picgo/b0959dc43e68202fd661c99a62082425.png)

## 3. 使用第三方清理工具

> 使用一些第三方工具的清理垃圾、软件搬家、软件卸载等功能,例如 `360安全卫士`等

- 1. 打开[360 官网](https://www.360.cn/)下载【`360安全卫士`】

![](/assets/picgo/6eac847cf628c1300a470214a25d61d7.png)

- 2. 360 安全卫士清理 C 盘垃圾步骤如下:

打开 360 安全卫士，上面的导航栏选择【电脑清理】,点击【清理 C 盘空间】按钮

![](/assets/picgo/b8235ef943d64958114bb589d3afac1d.png)

随后弹出一个 360 清理 pro 的窗口,在该窗口勾选清理内容,之后点击【扫描】按钮

![](/assets/picgo/ee24770ed8b9848d8b5dad655d0d6f62.png)

在打开的窗口勾选自己清理的选项,然后点击右侧下方的【立即清理】按钮。

![](/assets/picgo/9a0db2fb090c83b554379f684b80f81e.png)

随后弹出一个提示窗口,全选后再次点击【确定】按钮进行确认删除即可

![](/assets/picgo/1e7d77ffeb256c542b03d91e5bd21c96.png)

自动删除完成页面上就会显示清理完成,也可以再次尝试继续删除.

![](/assets/picgo/20815e552501e7e8a4bfccfd87ff448b.png)

## 总结

![](/assets/picgo/f4b1f1307ef820de996bf2abaa877f42.png)

本文总结了三种清理 C 盘电脑垃圾,解决了 C 盘爆满飘红的问题,清理临时文件,内置磁盘清理工具以及第三方清理工具.

通过上述方法,你可以有效地清理 C 盘垃圾文件,释放几十 G 的磁盘空间,保持系统的流畅正常运行.
